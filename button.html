BHN Button Styles

As we aim to keep all buttons across the site as consistent as possible. Any implementations of button classes should reflect their usage and follow the class naming system, detailed below:

Primary Button

btn-primary

Primary Button Outline

btn-primary-ol

btn-secondary selected

btn-secondary selected
